# Settings Screen
This project is a **Settings screen** using **Jetpack Compose** and **Material 3**.

The layout structure:

* A main `Column`
* Multiple `Row` components for each setting

  * **Left side:** Title + supporting text (`Column`)
  * **Right side:** Control (`Switch`, `Checkbox`, `Slider`, `Button`)
* Proper use of `Modifier.weight()` for alignment and spacing

---

## Material 3 Components Used

* `TopAppBar`
* `Scaffold`
* `Card`
* `Switch`
* `Checkbox`
* `Slider`
* `Button`
* `Divider`
* `AssistChip`
* `Snackbar`

---

## Modifiers Demonstrated

* `padding`
* `fillMaxWidth`
* `weight`
* `heightIn`
* `align`
* `border`
* `clip`
* `background`
* `clickable`

---

## Features

* Toggle notifications
* Enable dark mode
* Adjust volume (horizontal slider)
* Reset settings with Snackbar confirmation
* Display app version
